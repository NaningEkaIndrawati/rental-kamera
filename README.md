## Installation
**Install with Composer**
```
composer install
```
rename .env.example to .env, configure your env file

**Generate APP_KEY**
```
php artisan key:generate
```

**Databse Migration**
```
php artisan migrate
```

**Run**
```
php artisan serve
```
